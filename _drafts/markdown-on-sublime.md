steps:

1. install [package control](https://packagecontrol.io/installation)
2. 
        ctrl + shift + p
        "instal Package"
        "Markdown Extended && Markdown Previe"

3. edit your files... if want to preview,


        ctrl + shift + P
        "Markdown Preview" -> "Preview in browser"

4. be happy.
